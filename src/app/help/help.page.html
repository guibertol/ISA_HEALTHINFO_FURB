Ajuda

O que eu devo saber para cadastrar uma doença? você deve informar apenas o nome, a doença é representada pelo icone

O que eu devo saber para cadastrar um contato de emergência? você deve informar o nome, e-mail e telefone, tenha certeza dos dados corretos. o contato de emergência é representada pelo icone

O que eu devo saber para cadastrar um medicamento em uso? você deve informar o nome, tenha certeza de escrever de maneira correta, e lembre-se de remôver quando terminar o uso. o medicamento é representado pelo icone

O que eu devo saber para cadastrar uma alergia? você deve informar apenas o nome, a alergia é representada pelo icone

Qual o contato do desenvolvedor? você deve entrar em contato pelo email contato@furb.br ou atraves do telefone (47) 3321-0200

Quais dados o aplicativo armazena? o Aplicativo apenas armazena os dados inseridos atraves dos formularios de cadastro.
